# CODSOFT C++ Internship

This repository contains the C++ projects completed as part of my CODSOFT C++ Programming Internship.

## Tasks Completed

### Task 1: Simple Calculator
A simple calculator developed using C++ that performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division
 
### Task 2: Number Guessing Game
A C++ application that generates a random number and allows the user to guess it. The program provides hints whether the guessed number is too high or too low until the correct number is guessed.

### Task 3: To-Do List Manager
A task management application developed in C++ that allows users to:
- Add tasks
- View tasks
- Mark tasks as completed
- Manage daily activities efficiently
